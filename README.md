# LeetCode Solutions

Этот репозиторий содержит решения различных задач LeetCode. Каждый файл назван по номеру задачи, что облегчает поиск нужного решения.

## Structure

- Каждый файл с расширением `.cpp`, `.c`, `.rs`, `.js`, `.py`, `.ts`, `.cs`, или `.sql` содержит решение конкретной задачи LeetCode.
- Формат имени файла: `<problem_number>.<extension>` или `<problem_number>.<problem_title>.<extension>`, или `<problem_number> <problem_title>.<extension>`, например, `1452.cpp`, `1154.day-of-the-year.ts` или `75 Sort Colors.c`

## Languages

- C++ (`.cpp`)
- C (`.c`)
- Rust (`.rs`)
- JavaScript (`.js`)
- TypeScript (`.ts`)
- Python (`.py`)
- SQL (`.sql`)
- C# (`.cs`)
- Shell Script (`.sh`)

## Usage

Вы можете компилировать и запускать решения с помощью вашего любимого компилятора.

### For C++:
```sh
g++ 964.least-operators-to-express-number.cpp -o 964
./964
```

### For C:
```sh
gcc 3136.c -o 3136
./3136
```

### For Rust:
```sh
rustc 2409.rs
./2409
```

### For JavaScript:
```sh
node 2027.js
```

### For TypeScript:
```sh
ts-node 1154.day-of-the-year.ts
```

### For Python:
```sh
python 3446.py
```

### For SQL:
```sh
sqlite3 < 1824.minimum-sideway-jumps.sql
```

### For C#:
```sh
dotnet run 833.find-and-replace-in-string.cs
```
